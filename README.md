## ILI9341 to STM32 Pin Connections
| **ILI9341 Pin** | **STM32 Pin** |
|------------------|---------------|
| **VCC**          | 3.3V          |
| **GND**          | G             |
| **CS**           | B10/PB10           |
| **RESET**        | B1/PB1            |
| **D/C**          | B0/PB0            |
| **MOSI**         | A7/PA7            |
| **SCK**          | A5/PA5            |
| **LED**          | 3.3V          |
